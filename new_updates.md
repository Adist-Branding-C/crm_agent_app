# Recent Updates

## What new features have been added?
- Call log form and dialer intent support have been added so users can access calling functionality.
- Notification redirection and contextual routing were implemented, so app links now take users to the right place.
- Search suggestions, search history, and task completion toggling were added to improve search usability.
- A new search page was created and tested.

## Any new screens implemented?
- A search page screen was added to make search easier and more responsive.

## What issues or bugs were fixed?
- Lead deletion flow issues were fixed so deleting leads works more reliably.
- Basic issue context flow and state problems were fixed to improve app stability.

## Were there any significant code improvements or restructuring done?
- The app received architectural alignment and cleanup to make the code easier to maintain.
- Performance optimizations were made alongside clean code refactoring.
