# CRM Agent App

Lightweight Flutter CRM agent application providing lead and campaign management, analytics, and follow-up workflows.

## Features
- Lead creation and management
- Campaign and follow-up tracking
- Analytics dashboard
- Multi-platform: Android, iOS, web, desktop

## Prerequisites
- Flutter SDK (stable channel)
- Dart SDK (bundled with Flutter)
- Platform-specific tools for Android/iOS as needed (Android Studio, Xcode)

## Quick Start
1. Clone the repo:

   git clone <repo-url>
   cd crm_agent_app

2. Install dependencies:

   flutter pub get

3. Run on an attached device or emulator:

   flutter run

4. Run tests:

   flutter test

## Common Commands
- `flutter pub get` — fetch packages
- `flutter run` — run the app
- `flutter build apk` / `flutter build ios` — create release builds
- `flutter test` — run unit/widget tests

## Project Structure
- `lib/` — application source (blocs, screens, widgets, theme)
- `android/`, `ios/`, `web/` — platform folders
- `test/` — unit and widget tests
- `build/` — generated build outputs (ignored)

## Development Notes
- Keep `pubspec.yaml` up to date when adding packages.
- Follow existing folder conventions in `lib/` for new features.
